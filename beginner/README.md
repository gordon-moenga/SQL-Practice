# Easy SQL Practice
This folder contains my solutions and notes for the Easy SQL exercises completed on SQLPractice.
The purpose of this section is to strengthen my understanding of fundamental SQL concepts and develop confidence in writing basic queries for data retrieval and analysis.
## Topics Covered
- SELECT statements
- Filtering data with WHERE
- Sorting results with ORDER BY
- Limiting results
- Basic aggregate functions
  - COUNT()
  - SUM()
  - AVG()
  - MIN()
  - MAX()
- Working with NULL values
- Basic string functions
- Simple calculations in queries
## Structure
Each solution includes:
1. Problem statement
2. SQL solution
3. Query explanation
4. Concepts used
5. Key takeaways
## Learning Objectives
Through these exercises, I was able to:
- Understand the fundamentals of SQL querying
- Retrieve and filter data efficiently
- Perform basic data analysis using aggregate functions
- Improve logical and analytical thinking
- Build a strong foundation for intermediate and advanced SQL topics
